# 🎬 Netflix Data Analysis & Machine Learning Project

## 📌 Objective
The goal of this project is to analyze Netflix content data and uncover key trends related to content type, genre, release patterns, and ratings. Additionally, a machine learning model is built to classify content performance.

---

## 📊 Exploratory Data Analysis (EDA)
Performed data cleaning and analysis to answer key business questions:

- How has Netflix content grown over the years?
- What are the most popular genres?
- Which countries produce the most content?
- What is the distribution of Movies vs TV Shows?
- How does duration relate to ratings?

---

## 🔍 Key Insights

- 📈 Netflix content increased significantly after 2015
- 🎭 Drama and Comedy are the most dominant genres
- 🌍 Majority of content is produced in the US and India
- 🎬 Movies are more common than TV Shows
- ⏱️ Content duration has minimal impact on ratings

---

## 🤖 Machine Learning Model

- Built a classification model to categorize content into performance levels (High, Medium, Low)
- Used Scikit-learn for model building
- Achieved ~55% accuracy

### ⚠️ Challenges:
- Limited features in dataset
- Class imbalance affecting performance
- Scope for better feature engineering
